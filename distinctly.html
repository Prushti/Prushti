Intro

I work as a Product Designer at Distinct.ly, leading product strategy and design for Distinctly client and enterprise collaboration. Distinct.ly’s platform provides market research data and analysis based on clients demand and I have designed and developed many different projects for the company's small to medium business clients e-commerce websites, including a flower boutique, barber shop, vegan food products and grocery shop and many more.

SIMPLE SEARCH PROCESS

New dashboards can be intimidating due to the competitive market so I built this onboarding search experience to give the user a high-level overview of functionality and features. All users need to do is search their data by inputting a single word into the search bar.

CLEAR DATA VISUALIZATION

While designing this platform, I focused more on user experience rather than visual design. I used minimal color, clear CIA and understandable vision. I make it simple so that users can view market data analysis in a single view by consumer forensic. Each section on the overview page has horizontal scroll so that the user can view data easily without hassle. I put all the main functionality, like ROI Calculator, Targeting Audience, Channel recommendation and sentiment Analysis, in a single page so User can navigate easily.
